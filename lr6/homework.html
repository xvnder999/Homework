Задание 1 — Анализ угроз

▼

Описание: Составьте список из 10 актуальных киберугроз для организации (корпоративная сеть). Для каждой угрозы укажите тип, возможные последствия и метод противодействия.

Ожидаемый результат: Таблица угроз в формате PDF или DOCX.

  • Минимум 10 различных типов угроз
  • Классификация по STRIDE или другой методологии
  • Ссылки на источники (CVE, OWASP)
Оценка: 5/5 Проверено: 12.04.2025

Задание 2 — Настройка файервола

▼

Описание: Настройте правила iptables для защиты условного Linux-сервера. Разрешите только необходимые порты (SSH, HTTP, HTTPS), заблокируйте остальные.

Ожидаемый результат: Скрипт .sh с правилами и отчёт с объяснением каждого правила.

  • Файл скрипта iptables.sh с комментариями
  • Объяснение каждого правила
  • Скриншот вывода iptables -L после применения
Оценка: 5/5
✓ Выполнено

Задание 3 — Сетевое сканирование

▼

Описание: С помощью Nmap просканируйте учебный стенд (выданный адрес) и составьте отчёт о найденных открытых портах и сервисах.

Ожидаемый результат: Отчёт Nmap и его интерпретация.

  • Результат сканирования nmap -sV
  • Описание каждого открытого порта
  • Рекомендации по безопасности
Оценка: 4/5
✓ Выполнено

Задание 4 — Шифрование данных

▼

Описание: Реализуйте шифрование и дешифрование текстового файла с помощью алгоритма AES-256 на Python или Java. Продемонстрируйте работу программы.

  • Рабочий код с комментариями
  • Демонстрация шифрования и дешифрования
  • README с инструкцией запуска
Оценка: 5/5
⟳ На проверке

Задание 5 — Анализ веб-уязвимостей

▼

Описание: На учебном стенде DVWA найдите и задокументируйте уязвимости из OWASP Top 10: SQL-инъекцию, XSS, CSRF. Опишите каждую уязвимость и способ её закрытия.

  • Описание каждой найденной уязвимости
  • Скриншоты с демонстрацией эксплуатации
  • Рекомендации по устранению
Ожидает проверки
⟳ На проверке

Задание 6 — Аудит безопасности

▼

Описание: Проведите базовый аудит безопасности учебной системы. Используйте инструменты: Lynis, OpenVAS или аналоги. Составьте итоговый отчёт.

  • Автоматический отчёт аудита
  • Ручной анализ критических пунктов
  • План исправления уязвимостей (roadmap)
